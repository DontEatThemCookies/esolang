REGULAR BRAINFUCK INTERPRETER - Source

Version 1.2
This directory contains the RBFI source code (rbfi.py), as well
as a sample Hello World script in Brainfuck (script.bf).

Syntax: python3 rbfi.py [-h] [-d] [filename]
        -d is a flag for basic debugging.
        -h displays help.
Example: python3 rbfi.py script.bf
